== Description ==

Extra configuration settings for BuddyPress Admins.

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyPress Extended Settings` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page
5. Click Extended Settings link under the BuddyPress admin menu
6. Check the options you want

= Manual Installation =

1. Upload `buddypress-settings` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
5. Click Extended Settings link under the BuddyPress admin menu
6. Check the options you want
